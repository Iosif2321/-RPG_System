## Блок-схема древа навыков (ДРАКОН-стиль)

```mermaid
graph TD
    classDef startEnd fill:#1A1A2E,stroke:#3399FF,stroke-width:2px,color:#fff;

    Start([НАЧАЛО: Физиология → 0]):::startEnd --> CheckLevel{Уровень<br>физиологии ≥ 0}

    subgraph Tier1 [Уровень 0-5 - Тир 1]

    Tier7 --> End([КОНЕЦ: Физиология<br>максимальная]):::startEnd
```

### Легенда обозначений

| Обозначение | Значение |
|-------------|----------|
| ╔═══╗ | Основной блок / раздел |
| ┌───┐ | Информационный блок |
| ▼ | Стрелка потока (переход к следующему) |
| [номер] | Номер перка |
| full | Перк с реальным эффектом |
| chance | Перк с вероятностным эффектом |
