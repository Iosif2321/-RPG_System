### Элементы интерфейса

#### 1. **Верхняя панель**
- **Заголовок** — название системы
- **XP счетчик** — текущий опыт / опыт до следующего уровня
- **Уровень** — текущий уровень персонажа
- **Прогресс бар** — визуальное отображение прогресса (0-100%)

#### 2. **Панель статистики**
Отображает общую статистику игрока:
- `Убийства` — всего уничтожено врагов
- `Смерти` — количество смертей
- `Воскрешений` — поднято союзников
- `Ремонтов` — отремонтировано техники
- `Уничтожено техники` — подбито вражеских ТС
- `Время в игре` — общее время игры (часы)

#### 3. **Панель категорий XP**
Показывает накопленный XP по типам действий:
- `Медицина` — всего XP за лечение и воскрешение
- `Ремонт` — всего XP за ремонт техники
- `Бой` — всего XP за убийства и уничтожение техники
- `Поддержка` — всего XP за снабжение и заправку
- `Инженерия` — всего XP за инженерные действия

Каждая категория отображает:
- Название категории
- Общий XP в этой категории

> **Важно:** Категории носят **информационный характер** и не влияют на геймплей. Это просто статистика ваших действий.

#### 4. **Кнопки управления**
- `Обновить` — обновить данные в реальном времени
- `Закрыть` — закрыть меню (Esc также работает)

### Уведомления

#### Получение XP
```
┌─────────────────────┐
│  +50 XP (Убийство)  │  ← Всплывающее уведомление
└─────────────────────┘
```
- Появляется в правом верхнем углу
- Исчезает через 2 секунды
- Показывает полученное количество XP и причину

#### Повышение уровня
```
╔═══════════════════════════════╗
║    ПОВЫШЕНИЕ УРОВНЯ!          ║  ← Заголовок на весь экран
║                               ║
║    Вы достигли уровня 5!      ║  ← Сообщение
╚═══════════════════════════════╝
```
- Крупное уведомление на весь экран
- Звуковой сигнал
- Исчезает через 4 секунды

#### Повышение навыка
```
┌─────────────────────────────┐
│  + Навык: Медицина (3)      │  ← Уведомление о навыке
└─────────────────────────────┘
```
- Показывает название навыка и новый уровень
- Длительность 3 секунды

---

### Формулы расчета

#### Опыт до следующего уровня
```
XP(уровень) = 1000 × (уровень ^ 1.5)
```

Пример:
- Уровень 1 → 2: 1000 XP
- Уровень 2 → 3: 2828 XP
- Уровень 3 → 4: 5196 XP
- Уровень 4 → 5: 8000 XP

#### Прогресс до следующего уровня
```
Прогресс = (Текущий_XP - XP_предыдущего_уровня) / 
           (XP_следующего_уровня - XP_предыдущего_уровня)
```

### Примеры использования в консоли

```sqf
// Открыть меню
[] call RPG_fnc_openRPGMenu;

// Проверить XP
hint str ([player, "all"] call RPG_fnc_getXP);

// Проверить статистику по категории
private _data = [getPlayerUID player] call RPG_fnc_getPlayerData;
private _skills = _data get "skills";
hint str (_skills get "medical");  // XP за медицину

// Добавить XP (тест)
[player, 500, "Тест"] call RPG_fnc_addXP;
```
